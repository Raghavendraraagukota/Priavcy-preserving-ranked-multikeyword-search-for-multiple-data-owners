# Privacy Preserving Ranked Multi-Keyword Search for Multiple Data Owners in Cloud Computing

## Abstract
This project proposes a privacy-preserving searchable encryption framework enabling secure ranked multi-keyword search over encrypted cloud data shared by multiple data owners.

## Modules
### Data Owner
- File upload
- Encryption key generation
- Deduplication
- File sharing
- Security performance analysis

### End User
- Secure keyword search
- Ranked file retrieval
- Secret key request
- Secure download and decryption

### Cloud Server
- User/Owner authentication
- Audit monitoring
- File ranking management
- Throughput and delay analysis

### Attack Simulation
- Attacker detection
- Security auditing

## Features
- Ranked Multi-keyword Search
- Privacy Preserving Search
- Multiple Data Owners
- Secure Data Sharing
- Deduplication
- Attack Detection
- Cloud Auditing

## Tech Stack
- Java JSP/Servlet
- Apache Tomcat
- MySQL
- HTML/CSS/JavaScript
- Bouncy Castle Cryptography

1 Import into Eclipse Dynamic Web Project  
2 Configure MySQL database  
3 Deploy on Apache Tomcat
